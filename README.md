# Expense Tracker

## Project Description
This application helps you track your personal expenses. You can add expenses with amount and description, and view all recorded expenses.

## Features
- Add new expenses with amount and description
- View all expenses from database
- Simple and intuitive interface

## Database
The application uses a MySQL database called `expensetracker` with one table:
- **expenses**: Stores expense records with id, amount, and description
